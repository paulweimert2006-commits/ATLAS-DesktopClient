## 2026-02-25 - [Unified Fetch Button Pattern]
**Learning:** Combining multiple disparate technical fetch operations (BiPRO SOAP + IMAP) into a single, clearly labeled primary action ("Jetzt Dokumente abrufen") significantly reduces user anxiety and cognitive load in a complex technical interface. Using visual prominence (height + emoji) helps guide the user to the most frequent and safe workflow.
**Action:** Always look for opportunities to unify fragmented background processes into a single "Primary Workflow" button when the technical dependencies allow for parallel execution.
