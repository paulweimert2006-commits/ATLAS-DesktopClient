Hinweise zur Konfiguration

Gmail

  • Server: smtp.gmail.com
  • Port: 587
  • Aktivieren Sie "App-Passwörter" in Ihren Google-Kontoeinstellungen

Microsoft 365

  • Server: smtp.office365.com
  • Port: 587
  • Verwenden Sie Ihre vollständige E-Mail-Adresse als Benutzername

Sicherheit

Passwörter werden verschlüsselt in der Konfigurationsdatei gespeichert.
